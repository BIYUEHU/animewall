## Install

```bash
git clone https://github.com/BIYUEHU/animewall.git
```

打开`src/config.ts`文件

```typescript
export default {
	userName: 'himeno',
};
```

将`userName`更改为你的番组计划用户名或用户ID

> [Bangumi](https://bgm.tv/)

编译

```bash
npm run build
```
